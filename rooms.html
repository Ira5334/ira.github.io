Наші номери та ціни

У готелі "Горизонт" ми пропонуємо вам комфортні номери для будь-якого типу відпочинку. Ознайомтесь з нашим асортиментом номерів та цінами.

Економ-номер

Просторий номер для двох осіб, ідеальний для тих, хто шукає простоту і зручність.

Ціна: 800 грн/ніч

Стандарт-номер

Комфортабельний номер для двох осіб з усіма необхідними зручностями для відпочинку.

Ціна: 1200 грн/ніч

Люкс

Розкішний номер для двох осіб з окремою вітальнею та панорамними вікнами для насолоди краєвидами Карпат.

Ціна: 2500 грн/ніч

Сімейний номер

Номер для всієї родини з двома спальнями та простором для відпочинку дітей і дорослих.

Ціна: 2000 грн/ніч

Апартаменти

Розкішні апартаменти для комфортного відпочинку з усіма зручностями, включаючи кухню та балкон з видом на гори.

Ціна: 3500 грн/ніч
